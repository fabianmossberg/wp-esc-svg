
# WordPress esc_svg

Escape and display an inline svg.

## Installation

Download this repo as a zip, upload it to plugins in WordPress.

## `esc_svg()`

```php
esc_svg( string $inline_svg, $echo = false);
```
Escaping an svg image.

## `esc_svg_e()`

```php
esc_svg_e( string $inline_svg);
```

Display escaped svg.
